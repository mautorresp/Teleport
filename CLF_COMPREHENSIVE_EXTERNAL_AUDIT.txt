--------------------------------------------------
CLF (Causality Language Framework) implements a universal bijection between:
- Domain: Arbitrary byte sequences S of length L
- Codomain: LEB7-encoded parameters representing base-256 integers K
- Bijection: CBD256 universal mapping K ↔ S via bitstream reconstruction

KEY MATHEMATICAL PROPERTIES:
• Input S treated as big-endian base-256 integer: K = Σ S[i]·256^(L-1-i)
• LEB7 encoding: Variable-length base-128 with MSB-first 7-bit groups
• Minimal parameter: ceil(bitlen(K)/7) groups, no big-integer materialization
• Reconstruction: expand_cbd256_from_leb7(LEB7, L) → S via bitstream padding
