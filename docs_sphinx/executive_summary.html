Teleport CLF Calculator

Single-Seed Causal Logic Framework for Mathematical File Size Analysis

Cmin(1)(L) = 88 + 8 × leb(L)
Locked Constants: H=56, CAUS=27, END=5 | Decision Rule: EMIT ⟺ Cmin(1)(L) < 10×L (strict)

Mathematical Foundation

Core Formula: The CLF Calculator implements causal minimality using the formula Cmin(1)(L) = 88 + 8×leb(L), where:

  • 88 bits: Fixed overhead (H=56 + CAUS=27 + END=5)
  • 8×leb(L): Variable LEB128 encoding cost
  • leb(L): Unsigned LEB128 byte-length of L using 7-bit groups

Key Properties

  • Complexity: O(log L) computational complexity
  • Arithmetic: Integer-only operations (no floating-point)
  • Deterministic: Identical inputs produce identical outputs
  • Efficient: All practical file sizes result in EMIT=True
